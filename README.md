## Penjelasan Project
Acorn merupakan aplikasi Buku Tamu berbasis web Laravel yang digunakan untuk mencatat dan mengelola data pengunjung. Sistem ini memungkinkan pengguna untuk menginput data tamu, melihat daftar kunjungan, melakukan rekapitulasi berdasarkan rentang tanggal tertentu, serta mengekspor data tamu ke file Excel. Aplikasi ini dirancang untuk mempermudah proses pencatatan dan mengurangi ketergantungan pada pencatatan manual.

## Tech Stack
- Laravel
- PHP
- MySQL
- HTML, CSS, Bootstrap, Font Awesome
- Blade Template
- JavaScript
- Paket Export Excel Laravel

## Fitur-fitur Project
- Pencatatan data pengunjung (nama, alamat, tujuan, nomor HP, tanggal kunjungan)
- Login dan manajemen pengguna
- Filter dan rekapitulasi data berdasarkan tanggal tertentu
- Export data tamu ke file Excel
- Halaman pengisian data untuk tamu
